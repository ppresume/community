# Templates

PPResume provides several templates out of the box, including:

- [ModernCV Banking](./templates/moderncv-banking)
- [ModernCV Classic](./templates/moderncv-classic)
- [ModernCV Casual](./templates/moderncv-casual)

You can check [PPResume Gallery ->
Templates](https://ppresume.com/gallery/templates) for a quick preview.

Basically you can change the resume template by clicking the template name
button in the resume setting drawer:

![PPResume Switch Template](/static/images/typesetting/templates/ppresume-switch-template.webp)

Want to preview the template before you chant it? No problem! You can click on
the template thumbnail to zoom in and preview it:

![PPResume Preview Template](/static/images/typesetting/templates/ppresume-preview-template.webp)
![PPResume Preview Moderncv Casual](/static/images/typesetting/templates/ppresume-preview-moderncv-casual.webp)
